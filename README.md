# TuAdivinaNumero

Mi primer proyecto en C++.

Un juego donde la computadora genera un número aleatorio entre 1 y 100 y el usuario intenta adivinarlo.

## Funcionalidades actuales

- Generación de números aleatorios
- Sistema de intentos
- Pistas indicando si el número secreto es mayor o menor
- Validación de entradas incorrectas
- Manejo de errores de entrada
- Bucle de juego hasta acertar

## Conceptos aprendidos

- Variables y tipos de datos
- Condicionales (`if`, `else if`, `else`)
- Bucles (`do while`)
- Librería `random`
- Entrada de usuario con `cin`
- Validación de datos
- Uso básico de Git y GitHub
